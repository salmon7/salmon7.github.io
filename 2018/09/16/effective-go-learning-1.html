Go has no line length limit. DonвҖҷt worry about overflowing a punched card. If a line feels too long, wrap it and indent with an extra tab.

Switchпјҡ

  • caseеҗҺйқўеҸҜд»Ҙи·ҹеӨҡдёӘжқЎд»¶пјҢз”ЁйҖ—еҸ·еҲҶйҡ”еҚіеҸҜпјҢеҰӮ

  • дёҺcдёҚеҗҢзҡ„жҳҜпјҢgoзҡ„switchдёӯзҡ„caseеӯҗеҸҘе№¶дёҚдјҡеӣ дёәжІЎжңүbreakе°ұдёҖзӣҙжү§иЎҢпјҢиҖҢжҳҜеҸӘжү§иЎҢдёҖдёӘcase

  • е№¶дё”switchдёӯзҡ„breakжҳҜдёәдәҶжҸҗеүҚз»“жқҹcaseеҗҺзҡ„д»Јз ҒпјҢд»ҺиҖҢи·іиҪ¬еҲ°switchиҜӯеҸҘеқ—еҗҺ

  • еҰӮжһңswitchеӨ–жңүforеҫӘзҺҜпјҢеҲҷеҸҜд»ҘеңЁforеӨ–еўһеҠ labelпјҢbreak + labelеҲҷеңЁcaseдёӯеҸҜд»ҘзӣҙжҺҘи·іиҪ¬еҲ°еҫӘзҺҜеӨ–гҖӮеҪ“з„¶continueд№ҹеҸҜд»ҘдҪҝз”ЁlabelпјҢдҪҶжҳҜеҸӘеҜ№loopжңүз”Ё

  • switch д№ҹеҸҜд»Ҙз”ЁжқҘеҸ‘зҺ°дёҖдёӘжҺҘеҸЈеҸҳйҮҸзҡ„еҠЁжҖҒзұ»еһӢгҖӮйңҖиҰҒй…ҚеҗҲ type е…ій”®еӯ—дҪҝз”Ё
  • е®һйҷ…дёҠеЈ°жҳҺдәҶдёҖдёӘе…·жңүзӣёеҗҢеҗҚз§°дҪҶеңЁжҜҸз§Қжғ…еҶөдёӢе…·жңүдёҚеҗҢзұ»еһӢзҡ„ж–°еҸҳйҮҸпјҢеҰӮ
  • зұ»еһӢжҹҘиҜўпјҢе°ұжҳҜж №жҚ®еҸҳйҮҸпјҢжҹҘиҜўиҝҷдёӘеҸҳйҮҸзҡ„зұ»еһӢгҖӮдёәд»Җд№Ҳдјҡжңүиҝҷж ·зҡ„йңҖжұӮе‘ўпјҹgoalngдёӯжңүдёҖдёӘзү№ж®Ҡзҡ„зұ»еһӢinterface{}пјҢиҝҷдёӘзұ»еһӢеҸҜд»Ҙиў«д»»дҪ•зұ»еһӢзҡ„еҸҳйҮҸиөӢеҖјпјҢеҰӮжһңжғіиҰҒзҹҘйҒ“еҲ°еә•жҳҜе“ӘдёӘзұ»еһӢзҡ„еҸҳйҮҸиөӢеҖјз»ҷдәҶinterface{}зұ»еһӢеҸҳйҮҸпјҢе°ұйңҖиҰҒдҪҝз”Ёзұ»еһӢжҹҘиҜўжқҘи§ЈеҶіиҝҷдёӘйңҖжұӮ

Functionsпјҡ

Multiple return valuesпјҡ

  • еҸҜд»Ҙиҝ”еӣһеӨҡдёӘеҖјпјҢйҒҝе…ҚдәҶзұ»дјјCзҡ„дј жҢҮй’ҲеҲ°еҮҪж•°дёӯжүҚиғҪиҜҘж•°еҖјзҡ„ж–№ејҸ

Named result parametersпјҡ

  • е®ҡд№үеҮҪж•°зҡ„иҝ”еӣһеҖјзұ»еһӢзҡ„ж—¶еҖҷпјҢд№ҹеҸҜд»ҘжҢҮе®ҡеҸҳйҮҸеҗҚ
  • жҢҮе®ҡдәҶеҸҳйҮҸеҗҚеҗҺпјҢеҪ“еҮҪж•°ејҖе§Ӣж—¶пјҢе°Ҷдјҡж №жҚ®е®ғд»¬зҡ„зұ»еһӢиҝӣиЎҢйӣ¶еҖјеҲқе§ӢеҢ–пјҢиҝҷж—¶еҮҪж•°еҸҜд»ҘзӣҙжҺҘreturnдёҚз”Ёж·»еҠ д»»дҪ•еҖјпјҢдјҡй»ҳи®Өиҝ”еӣһе·Із»ҸеЈ°жҳҺзҡ„иҝ”еӣһеҸҳйҮҸ

Deferпјҡ

  • еңЁи§Јй”Ғдә’ж–Ҙй”Ғе’Ңе…ій—ӯж–Үд»¶дёӯжңҖеёёз”Ё
  • еңЁеҮҪж•°иҝ”еӣһеүҚпјҢз«ӢеҲ»и°ғз”Ёиў«еЈ°жҳҺдёәdeferзҡ„еҮҪж•°
  • еҰӮжһңиў«deferзҡ„еҮҪж•°жңүеҸӮж•°пјҢйӮЈд№Ҳ еҸӮж•°еҖјдёәdeferиҜӯеҸҘжү§иЎҢж—¶еҸӮж•°зҡ„еҖјпјҢдёҚдјҡеӣ дёәеҸӮж•°еңЁеҗҺйқўзҡ„жөҒзЁӢдёӯиў«жӣҙж”№иҖҢеҜјиҮҙdeferеҮҪж•°зҡ„еҸӮж•°иў«дҝ®ж”№
  • еҰӮжһңжңүеӨҡдёӘdeferж—¶пјҢйҒөеҫӘвҖңLIFOвҖқеҗҺиҝӣе…ҲеҮәзҡ„еҺҹеҲҷпјҢе…Ҙж ҲеҮәж Ҳ

Dataпјҡ

Allocation with newпјҡ

  • goжңүдёӨдёӘеҲҶй…ҚиҜӯеҸҘпјҢnewе’Ңmake
  • new
    • еҶ…е»әеҮҪж•°
    • еҲҶй…ҚеҶ…еӯҳпјҢдҪҶжҳҜдёҚеғҸе…¶д»–иҜӯиЁҖдёҖж ·еҲқе§ӢеҢ–еҶ…еӯҳпјҢиҖҢжҳҜд»…д»…з”Ёйӣ¶еҖјеЎ«е……е®ғ
    • new(T)еңЁеҶ…еӯҳеҲҶй…ҚдәҶдёҖдёӘйӣ¶еҢ–зҡ„TпјҢе№¶иҝ”еӣһTзҡ„жҢҮй’Ҳ *T
  • з”ұдәҺиҝ”еӣһзҡ„еҶ…еӯҳnewдёәйӣ¶пјҢеӣ жӯӨйңҖиҰҒйӣ¶еҖјзҡ„ж•°жҚ®з»“жһ„жғ…еҶөдёӢпјҢеҸҜд»ҘзӣҙжҺҘдҪҝз”ЁдёҚз”ЁиҝӣдёҖжӯҘзҡ„еҲқе§ӢеҢ–
  • йӣ¶еҖјжңүдј йҖ’жҖ§

Constructors and composite literals:

  • еҰӮжһңдёҚз”ЁеҲқе§ӢеҢ–з»“жһ„дҪ“зҡ„еҶ…йғЁеӯ—ж®өпјҢеҲҷ new(File) зӯүд»·дәҺ &File{}
  • еҸҜд»ҘжҢҮе®ҡеҲқе§ӢеҢ–Fileзҡ„жҹҗдәӣеӯ—ж®өпјҢеҰӮ &File{fd: fd, name: name}

Allocation with make:

  • дёҺnewзӣёжҜ”пјҢеҶ…зҪ®еҮҪж•°make(T, args)жңүдёҚеҗҢзҡ„з”ЁйҖ”new(T)гҖӮе®ғд»…еҲӣе»әsliceпјҢmapsе’ҢchannelsпјҢе№¶иҝ”еӣһзұ»еһӢеҲқе§ӢеҢ–пјҲйқһйӣ¶еҖјпјүзҡ„TпјҲдёҚжҳҜ*TпјүгҖӮ
  • жңүиҝҷж ·зҡ„еҢәеҲ«зҡ„еҺҹеӣ жҳҜпјҢиҝҷдёүз§Қзұ»еһӢзҡ„ж•°жҚ®з»“жһ„еҝ…йЎ»еҲқе§ӢеҢ–дәҶжүҚиғҪдҪҝз”ЁпјҢжҜ”еҰӮsliceеҝ…йЎ»еҲқе§ӢеҢ–жҢҮеҗ‘ж•°з»„зҡ„жҢҮй’ҲгҖҒй•ҝеәҰгҖҒе®№йҮҸпјҢmapе’Ңchannelд№ҹжҳҜеҰӮжӯӨгҖӮ
  • жҜ”еҰӮпјҢmake([]int,10,100)пјҢеҲҶй…ҚдёҖдёӘ100дёӘж•ҙж•°зҡ„ж•°з»„пјҢз„¶еҗҺеҲӣе»әдёҖдёӘй•ҝеәҰдёә10дё”е®№йҮҸдёә100зҡ„еҲҮзүҮз»“жһ„пјҢжҢҮеҗ‘ж•°з»„зҡ„еүҚ10дёӘе…ғзҙ гҖӮжіЁж„Ҹеә•еұӮжҳҜз”іиҜ·capеӨ§е°Ҹзҡ„ж•°з»„пјҢиҖҢ c=new([]int) еҲҷиҝ”еӣһдёҖдёӘжҢҮеҗ‘ж–°еҲҶй…Қsliceж•°жҚ®з»“жһ„пјҢиҷҪз„¶lenе’ҢcapдјҡеҲқе§ӢеҢ–дёә0пјҢ*дҪҶжҳҜиҝҷдёӘsliceзҡ„жҢҮй’ҲжҢҮеҗ‘nilпјҢc ==nil дёәtrueгҖӮ
  • д»ҘдёӢдҫӢеӯҗи§ЈйҮҠдәҶnewе’Ңmakeзҡ„еҢәеҲ«гҖӮ

Arrays:

  • ж•°з»„еңЁGoе’ҢCдёӯзҡ„е·ҘдҪңж–№ејҸжңүеҫҲеӨ§е·®ејӮгҖӮеңЁGoдёӯпјҢ

    • зү№еҲ«жҳҜпјҢеҰӮжһңе°Ҷж•°з»„дј йҖ’з»ҷеҮҪж•°пјҢе®ғе°ҶжҺҘж”¶ж•°з»„зҡ„еүҜжң¬пјҢиҖҢдёҚжҳҜжҢҮеҗ‘е®ғзҡ„жҢҮй’ҲгҖӮ
    • ж•°з»„зҡ„еӨ§е°ҸжҳҜе…¶зұ»еһӢзҡ„дёҖйғЁеҲҶгҖӮзұ»еһӢ[10]int е’Ң[20]intдёҚеҗҢ
  • еҪ“з„¶еҸҜд»Ҙз”Ё & жҳҫејҸең°еҸ–ж•°з»„зҡ„ең°еқҖпјҢеҰӮ
